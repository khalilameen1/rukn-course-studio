## Instruction precedence

Resolve conflicts in this order

1. Safety verified domain truth and current authoritative requirements
2. Explicit user product decisions audience decisions and output constraints
3. This standard
4. Verified course specific source material
5. Reference creator style calibration
6. Unverified user or source claims treated as hypotheses
7. Model defaults

Style references never override facts
User supplied factual claims never override verified truth merely because they came from the user
Source prose never overrides the requested spoken language
Market conventions never justify inaccurate unsafe or unlawful teaching
Default counts never override the capability needed to fulfil the course promise

## Definition of success

The output succeeds only when all of the following are true

- The stated learner can enter without hidden prerequisites
- A learner who completes the practice can perform the promised real task at the stated level
- Every lesson changes an action decision diagnosis or mental model
- The course contains domain depth unavailable from generic social media summaries
- The language sounds like a credible practitioner speaking naturally to the target learner
- A final independent spoken variety pass has removed written register residue without changing accepted meaning
- Terms match how capable people in the target market actually speak
- Tool teaching is subordinate to principles and remains useful when tools change
- Projects create evidence of capability rather than decorative homework
- The course progresses toward a genuine peak and professional deliverable
- Short drama mechanics are present only as invisible causal progression earned tension complete lesson payoffs and accumulating capability
- The instructor remains a calm credible practitioner rather than performing an entertainer persona or manufacturing suspense
- The exported document contains only the lecturer speech and allowed headings or project boundaries
- Teleprompter layout is visually readable sentence by sentence
- No fatal quality gate remains open
- The delivered course is the result of completed internal drafting review rewrite and rendering passes rather than an exposed first draft

Passing structural validators without satisfying these conditions is failure

## Version 1.3 change

This version adds a mandatory spoken variety integrity gate after all meaning and course wide rewrites
It reviews syntax morphology connectors word order and auditory habits rather than searching for isolated words only
Every language repair must preserve the accepted claim condition exception sequence contrast example action and continuation dependency
Any language edit invalidates the earlier render and requires affected meaning gates teleprompter checks and rendering to run again
